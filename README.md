# tv-remote-cli
> CLI for remotely controlling your Smart TV

---

## Installation

```bash
npm install -g tv-remote-cli
```

## Usage

For list of all commands type `help`.

Example:

```
‚ùØ tv-remote
TV$ connect 192.168.1.126

TV$ volup
TV$ Samsung Remote Client: disconnected from 192.168.1.126:55000
TV$ voldown
TV$ Samsung Remote Client: disconnected from 192.168.1.126:55000
TV$ menu
TV$ Samsung Remote Client: disconnected from 192.168.1.126:55000
TV$ return
TV$ Samsung Remote Client: disconnected from 192.168.1.126:55000
TV$ source
TV$ Samsung Remote Client: disconnected from 192.168.1.126:55000

TV$ down
TV$ Samsung Remote Client: disconnected from 192.168.1.126:55000
TV$ enter

TV$ exit
```
